# LED Chess Board

An interactive 8x8 LED chess board that allows users to play against an AI opponent. Built with Arduino Mega and WS2812B addressable LEDs.

## Features
- Full chess gameplay with move validation
- AI opponent using Micro-Max chess engine
- Visual feedback via LED colors
- Serial monitor interface for move input

## Hardware
- Arduino Mega 2560
- 64 WS2812B LEDs
- Custom PCB/breadboard layout
